# 🎃 Slashing Pumpkins

The official website for the **Slashing Pumpkins**, a novice-level beer-league ice hockey team based in **Fairfax, Virginia**.

🔗 **[fairfaxslashingpumpkins.com](https://fairfaxslashingpumpkins.com)**

The site is the team's public-facing hub — check out the roster and player profiles, keep up with the schedule and game results, learn about the team, and find links to where we hang out online. It's built mobile-first, since that's where most of our fans are.

## 🛠️ Built With

- **Vue 3** (Composition API) + **TypeScript**
- **Vite** — build tooling and dev server
- **Vue Router** — client-side routing
- **PrimeVue** — UI components
- **Tailwind CSS** — styling
- **ESLint** + **Prettier** — code quality and formatting
- **Netlify** — hosting and deployment
- **Plausible.io** — privacy-friendly analytics

## 🚀 Getting Started

```sh
npm install   # install dependencies
npm run dev   # start the local dev server
```
